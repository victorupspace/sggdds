# Architecture

O pacote e ESM-first, TypeScript strict e preparado para publicacao privada futura.

## Decisoes

- `src/index.ts` e o ponto de entrada publico do pacote.
- `.storybook` nao deve ser importado pelo codigo de producao.
- `src/foundations` concentra tokens e fundamentos tecnicos.
- `src/components` recebera componentes React um por vez.
- `platforms` documenta a evolucao multiplataforma sem introduzir Swift ou Kotlin agora.
- CSS deve consumir custom properties geradas por tokens, evitando acoplamento a frameworks utilitarios.

## Governanca

Mudancas devem ser pequenas, revisaveis e acompanhadas por docs ou testes quando alterarem comportamento publico.
